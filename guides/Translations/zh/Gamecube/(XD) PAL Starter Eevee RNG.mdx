## 所需工具

- 支持 Lua 的 Dolphin 模拟器（可在相关 Discord 获取）
- GameCube 专用 Lua 脚本（可在相关 Discord 获取）
- [PokeFinder](/pokefinder)
- PAL-Pal 工具（由 Kapital#9267 开发）(https://github.com/KapitalRoser/PAL-Pal)
- 一个全新的 PAL 版 XD 存档
- Parzival#3035 整理的异色伊布个体值种子列表（适用于方法 2）(https://pastebin.com/0Dh740Kg)
- RunAsDate（64 位）工具，用于方法 2

## 简介

本指南介绍了两种在 PAL 版宝可梦 XD 中乱数初始伊布的方法。
根据你的需求选择使用哪一种：

- 方法 1：只使用 PAL-Pal 工具，适合找任意一只伊布（但不可预知性较高，个体可能不好）
- 方法 2：使用已知的异色伊布种子列表，通过初始种子乱数获取高素质个体

本指南默认你已了解基本的模拟器设置及 PAL XD 的初始种子乱数方法。

```
注意：本指南只适用于 PAL 版的 XD，不适用于 NTSC（美版）！
```

## 帧数推进方法说明

由于 PAL 版在命名界面时 RNG 会冻结，你需要以下方式推进帧数：

1. 重选宝可梦：主菜单中选择“VS 模式” -> “快速对战” -> “与 CPU 对战” -> 终极。每次出现宝可梦画面时会推进一次帧数。按“否”并重复。
2. 记忆卡读取：在主菜单中按 B 和 A，会重新加载记忆卡数据，画面会提示，此操作推进 1 次。
3. 设置保存：在设置中切换震动选项并保存，然后返回主菜单。
4. 命名界面黑屏法：开始新游戏时，当询问是否使用特定名字时按 B 退出。不要选择名字，否则无法回退。

```
注意：只有当当前种子等于目标种子时，才设置训练师名字。否则请勿继续进入命名界面！
```

## 方法一：仅使用 PAL-Pal 工具

这是最简单的方法，适合乱数任意伊布（可能无法获得理想个体，耗时也可能较长）：

1. 打开 Dolphin，加载 Lua 脚本，在主菜单查看当前种子，然后暂停模拟器。
2. 打开 PAL-Pal，进入「Speedrun」标签页，设置筛选条件（是否异色、个体、性别、性格），点击保存。
3. 将当前种子（Hex）输入 PAL-Pal 的对应栏位，搜索结果。
4. 通过「Next」和「Previous」按钮浏览结果。
5. 找到目标结果后，按说明进行推进，完成乱数。

## 方法二：使用异色伊布种子列表

此方法需要进行初始种子乱数，但能快速获得高个体或理想性格的异色伊布。

1. 打开 PokeFinder，进入「Gamecube RTC」标签页。将想要的异色伊布的种子输入 Target Seed 栏位。请确保初始种子位于推进 15,000 - 20,000 的区间。
2. 使用 RunAsDate、Dolphin 和 Lua 脚本完成初始种子乱数。在主菜单暂停模拟器。
3. 打开 PAL-Pal，进入「Custom」标签页，输入当前种子。
4. 目标标题种子（Target Title Seed）并不是你想要的最终种子，而是用于命中它的中间种子。打开 PokeFinder 的 Researcher，模式设为 XDRNG[R]，将目标种子填入 Seed 栏位，设置最大推进为 1010，点击生成。查看推进 1000 处的种子，即为 Target Title Seed。
5. 在 PAL-Pal 中输入该种子并点击 Search，你的目标伊布应该会出现在结果第一行。按提示推进种子并获取伊布。

```
注意：实际延迟可能不是刚好 1000，可能略微有出入。建议先按 1000 尝试，如果不符，则以帧数 1002 或 1004 的结果为准继续微调推进。
```
