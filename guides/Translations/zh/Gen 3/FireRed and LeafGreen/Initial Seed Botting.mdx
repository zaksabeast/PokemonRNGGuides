### 所需工具：

- [带有 Lua 脚本的 mGBA 模拟器](/mgba-setup)
- [PokeFinder](/pokefinder)
- 由 Real96 开发的 [FRLGRSEInitialSeedsFinder](https://github.com/Real96/FRLGRSEInitialSeedsFinder)

## 简介

本指南介绍一种适用于《火红》《叶绿》的替代乱数方式，借助 Lua 自动脚本实现对初始种子的更高控制。

### 设置步骤：

1. 首先确定你的目标。在 PokeFinder 中进入 第三世代 => 定点乱数 或 野生乱数（两者操作方式相似）。
2. 在 "检索器" 标签页中设置所需的筛选条件。
3. 生成结果后，记录你选中的种子。
4. 运行 FRLGRSEInitialSeedsFinder 工具。它会要求输入一个种子，此时输入刚刚记录的那个。
5. 当工具提示“请输入想生成的结果数量”时，建议输入一个较大的数字，例如 100。
6. 等待工具生成结果，它将输出一个文本文件，列出所有可能的初始种子。打开该文件并复制全部内容。
7. 打开 FRLG 的 Lua 脚本，找到如下行： 'local botTargetInitSeeds ='。将你复制的种子内容以 {} 括起来并填入该行。
8. 保存脚本。

```
对于刚开始游戏、尚未获得教学电视的初始 ID 乱数，建议只考虑较低帧范围内的结果。
```

### 自动执行流程：

1. 编辑 Lua 脚本的第一行。
2. 进入游戏的继续画面（读取存档的界面）。
3. 同时按下电脑的 Shift 键和 GBA 的 SELECT 键启动自动脚本。

一旦脚本完成命中，游戏将自动暂停。此时记录下你命中的初始种子，在 PokeFinder 中输入该种子后续进行乱数操作。你现在可以确定当前帧可用于继续乱数流程。
