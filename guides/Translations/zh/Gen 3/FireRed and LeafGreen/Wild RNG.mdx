## 所需工具

- [带有 Lua 脚本的 mGBA 模拟器](/mgba-setup)
- [PokeFinder](/pokefinder)
- 表 ID 和里 ID（若目标为异色宝可梦）

```
提示：如果你只是为了异色宝可梦而进行乱数，请确保在 PokeFinder 中设置含有该存档 TID 与 SID 的配置档。
```

## 第一步：设置 PokeFinder

1. 打开 PokeFinder，选择第三世代的 “野生乱数”，并确保你位于 “生成器” 标签页。
2. 将 “方式” 设置为 方式 1。

```
提示：《火红·叶绿》中确实存在其他方法，但 H-1 是最常见、最推荐的。其他方法依赖于 v-blank，出现几率极低。
```

3. 将“地点” 设置为你将要进行乱数的地点。
4. 如果你有指定目标宝可梦，在选择地点后可在 “宝可梦” 中选择该宝可梦。
   - 若宝可梦未出现在列表中，或者你想要指定的遭遇槽，请选择 “遇敌种类” 中对应的数字，并将 “宝可梦” 留空。
   - 可参考 [此网站](https://sites.google.com/site/pokemonslots/gen-iii/emerald)获取具体遭遇槽数据：
5. “遇敌” 设置为 “草” 或 “冲浪”，本指南聚焦于使用甜甜香气触发战斗。

```
提示：《火红·叶绿》中迷人之躯和同步并不会影响乱数。
```

6. 填写目标地点的延迟值。
   - 可参考 [Google Doc](https://docs.google.com/spreadsheets/d/1cVweVvJXCXeTZOBVKVCBbcSI46rqBXV3ahbuoSGOnzk/edit#gid=1091733147) 获取每个区域的延迟值。
   - 注意：通常存在 ±1 的误差范围。

## 第二步：获取初始种子

1. 在你想进行乱数的地点存档。
2. 重启模拟器。
3. 游戏加载后，进入继续画面。
4. 暂停游戏，此时 Lua 脚本会在屏幕上显示初始种子。

```
提示：《火红·叶绿》每次加载游戏都会生成不同的种子，和《红蓝宝石》无电电池或《绿宝石》的固定种子不同。这意味着你可以反复重置来刷更理想的种子。
```

![初始种子](/images/FireRed-LeafGreen/Wild/Initial-Seed.png)

## 第三步：寻找目标帧数

1. 在 Seed 输入框中填写初始种子。
2. 设定目标宝可梦的条件（是否异色、个体值、性格等）。
   - 如果目标为完美个体的异色，可能需要等待非常高的帧数。
   - 若无结果，请放宽筛选条件。
3. 点击 “生成”。

![Setup](/images/FireRed-LeafGreen/Wild/Setup.png)

如图所示，本例的筛选条件是“任意异色宝可梦（Route 5）”。

```
提示：你可以按 B 键回到开场动画，再按 A 进入继续画面，借此更换初始种子。重复操作直到得到一个更合适的种子。
```

## 第四步：开始乱数

1. 找到目标帧数后，进入游戏。
2. 推进到接近目标帧数前几千帧的位置。

```
提示：教学电视可以极大地加速推进帧数。使用它可使乱数每帧推进速度提升 313 倍，可在几分钟内推进数百万帧。

在背包中打开教学电视并关闭它，即可迅速推进帧数。
```

3. 打开训练师卡片并按下 A 翻转 一次。
   - 翻转能刷新延迟，有助于避免因帧数过高而导致延迟偏移。
4. 选择拥有“甜甜香气”的宝可梦，移动光标至该技能上。
5. 当接近目标帧数时，暂停游戏并建立即时存档。然后使用 Ctrl + N 逐帧推进至目标帧。
6. 在目标帧时，同时按住 A 并取消暂停。
   - 如果遇到的宝可梦不是目标个体，请尝试在前后各一帧进行操作。

若上述操作正确无误，你将成功遇到目标宝可梦。完成了第三世代的野生乱数！

![成功！](/images/FireRed-LeafGreen/Wild/Success.png)

## 问题排除

如果尝试了 ±1 帧仍未命中目标，请手动判断你实际命中的帧数，并据此调整。
