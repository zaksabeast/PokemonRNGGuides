## 所需工具

- [PokeFinder](/pokefinder)
- [包含lua脚本功能的Desmume模拟器](/desmume-setup)

## 准备工作

1. 打开 PokeFinder -> 第四世代 -> 定点 -> 检索器，在设置中选择你想要的宝可梦。
2. 设置或选择正确的个人资料，并根据需要调整筛选条件。
3. 搜索目标种子以及目标帧。

## 初始种子乱数

请参考[此指南](/dppt-initial-seed)来进行初始种子乱数。

## 推进帧数

请参考[此指南](/dppt-advance-rng)来推进帧数。

## 校准

1. 在 PokeFinder 中进入生成器选项卡。
2. 输入初始种子并找到你的目标宝可梦的帧数。
3. 搜索你实际遇到宝可梦的帧数。
4. 根据帧数调整延迟并重试。

```
目标帧 - 实际命中帧 = 延迟
```

## 特殊情况

如果想在毁坏的世界乱数骑拉帝纳，会比较棘手。进入毁坏的世界时，RNG 会推进 11 次，在遇到骑拉帝纳时总共推进 12 次。因此，如果使用日记推进法，至少需要考虑 23 次的初始推进。

在钻石和珍珠中乱数御三家时，建议在进入湖泊前保存游戏，并使用 0 帧的异色目标会更容易。在白金中，由于干扰更大，难度较高。

克雷色利亚和艾姆利多在首次对话时生成，因此需要在它们逃跑前进行乱数。

与大木博士对话领取关都三圣鸟时，不同宝可梦的初始帧不同：火焰鸟为 1、闪电鸟为 6、急冻鸟为 11。建议使用四天王方法，即先打败目标宝可梦，再与大木博士对话进行乱数。

## 玛纳霏蛋乱数

玛纳霏蛋的领取方式类似于神秘礼物宝可梦，但它的生成方式使用 Method 1。神秘礼物的乱数遵循常规规则。

玛纳霏有异色锁定，无法在获得它的游戏中孵出异色形态。要绕过这一限制，可以在不同游戏中进行乱数操作：

1. 确定游戏 A 的异色 PID。
2. 在游戏 B 中获得与该 PID 匹配的蛋。
3. 将蛋从游戏 B 交换到游戏 A。
4. 在游戏 A 中孵化蛋。

恭喜，你现在拥有一只异色玛纳霏了！
