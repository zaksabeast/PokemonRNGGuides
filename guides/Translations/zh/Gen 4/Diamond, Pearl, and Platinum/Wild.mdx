## 介绍

本指南假设你已经掌握了第四世代乱数的基础知识，如命中初始种子和推进帧数。

## 通用准备

在进行具体乱数前，先设置[PokeFinder](/pokefinder)并确定目标帧：

1. 打开PokeFinder => 第四世代 => 野生乱数 => 检索器。
2. 建立你的个人资料，选择好个人资料之后，根据你的目标填写好设定项（请注意宝可梦的出现时间）。
3. 设置好筛选条件之后计算结果（若无结果请减少筛选条件），选择你的目标。

## 基础野生乱数

你需要在目标区域使用甜甜香气（需要学会该技能的宝可梦）或甜甜蜜（花苑花田购入）进行野生遭遇：

1. 命中初始种子。
2. 打开宝可梦队伍菜单。
3. 推进帧数。
4. 使用甜甜香气或甜甜蜜。
5. 不要退出菜单，以防止随机数干扰。

与第五世代不同，甜甜香气和甜甜蜜的动画效果不会影响帧数推进。

## GBA卡带联动遭遇

操作流程与野生遭遇相同，正确配置 PokeFinder 后，工具会自动显示相关信息。

## 钓鱼乱数

```
注意：任何所谓的“提高钓鱼成功率的方法”都无效，PokeFinder 已经考虑了 DPPt 的相关 bug。
```

1. 前往目标水域。
2. 在宝可梦队伍菜单内推进帧数。
3. 打开背包并使用钓竿。

模拟器用户提示：若乱数帧数大幅推进，表明成功上钩并可能命中目标遭遇。PokeFinder 会显示可遭遇的帧数范围。

### 丑丑鱼乱数

丑丑鱼的遭遇机制是特殊的，步骤如下:

1. 先确认正确的丑丑鱼分布水域。
2. 存档并设置乱数参数。
3. 使用 PokeFinder 生成器输入初始种子，找到目标帧。
4. 确保目标帧能够触发遇敌，否则可能会遇不到任何东西。

当前丑丑鱼的随机性机制尚未完全破解，成功率可能波动。

## 自豪的后院乱数

1. 在庭院主人面前存档。
2. 打开PokeFinder => 第四世代 => 野生乱数 => 检索器。选择自豪的后院（Trophy Garden）。
3. 筛选你想要的宝可梦，并勾选遇敌种类为6（想要更多结果则为6、7），计算结果选择目标种子。
4. 以任意初始种子重新加载游戏。打开 PokeFinder 选择定点乱数输入当前种子生成结果。
5. 对照下表，寻找PID第五位数值相对应的帧数，推进至目标帧。
6. 与庭院主人对话确定当日宝可梦，随后存档。
7. 命中目标种子，就像普通野生乱数那样进行下去。

| :-- | -------------------- |
| 0   | 伊布                 |
| 1   | 胡说盆栽             |
| 2   | 小福蛋               |
| 3   | 喵喵                 |
| 4   | 皮宝宝               |
| 5   | 皮可西               |
| 6   | 宝宝丁               |
| 7   | 正电拍拍             |
| 8   | 胖丁                 |
| 9   | 3D龙 (DP)/百变怪(Pt) |
| A   | 漂浮泡泡             |
| B   | 负电拍拍             |
| C   | 魔尼尼               |
| D   | 玛力露               |
| E   | 吉利蛋               |
| F   | 玛力露丽             |

```
注意：不同工具可能导致索引偏差 ±1，必要时需手动调整。

```

## 大湿地乱数

大湿地的乱数类似于自豪的后院，其遭遇槽6 和 7 也存储了每日宝可梦。

1. 在眺望台选择目标宝可梦。
2. 在大湿地入口前保存游戏。
3. 选择目标帧范围，避免走动导致额外乱数干扰。
4. 按普通野生乱数方式进行。
