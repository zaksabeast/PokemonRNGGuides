## 所需工具

- [Desmume](https://www.pokemonrng.com/desmume-setup)
- [RNG Reporter](https://github.com/Bambo-Rambo/RNGReporter)

## 第一步：寻找个体组合

1. 在 RNG Reporter 中，将月份设置为 2 月或 11 月。
2. 将遭遇类型设为「Roaming Pokémon」，方式选择「IVs (Standard Seed)」。
3. 选择一个理想的个体值组合，此时暂时不用考虑异色。

![Setup](/images/Black-and-White/Roamer/Setup.png)

```
注意：如果你选择其他月份，帧数会非常不稳定，完全靠运气。
```

```
提示：生成目标组合可能会花上一段时间。
```

![目标种子](/images/Black-and-White/Roamer/Target.png)

## 第二步：获取 PID 与性格

刚才我们只是筛选了个体值组合，因为建议先通过 TID/SID 乱数获取一个适合异色的 ID。

```
如果你只想要高个体值或只想要异色，而不是两者兼得，可以跳过这一步和下一步。
```

选择一个帧数（建议至少 1500 帧，如果是 2 月或 11 月以外的月份则建议 2000 以上），并记录下目标 PID 与帧数。

![Pandora's Box](/images/Black-and-White/Roamer/Pandora.png)

## 第三步：准备存档

和第三、第四世代一样，推荐使用 ID 乱数（TID/SID）先拿到一个适合的异色 ID。若只在意个体或异色可跳过此步骤。

1. 选择你喜欢的 TID/SID 组合（或指定一个组合）并进行乱数。
2. 命中初始种子后，推进帧数。
3. 记下你获得的 TID/SID，推进到可以释放游走宝可梦的剧情处。
4. 然后在屋内存档。

你应该在这里存档（7号道路的小屋）：

![存档地点](/images/Black-and-White/Roamer/Save.png)

## 第四步：校准

![判定界面](/images/Black-and-White/Roamer/Final-Screen.png)

RNG Reporter 中会提供一个帧数，通常是 1 到 6 之间：

1. 在屋子里通过走动来推进个体帧数。
2. 建立即时存档，然后走出屋外，到达如上图所示的画面。
3. 开始推进性格帧数。
4. 到达目标帧数时按下 `A` 键。

```
注意：第一次大概率不会命中目标帧数。你可以查看图鉴确认是否成功（前提是你在追求异色）。
```

5. 遇到游走宝可梦后，记录它的 PID。你可以使用 Lua 脚本查看，或直接捕捉后用 PkHeX 查看数据。

![校准](/images/Black-and-White/Roamer/Calibration.png)

## 第五步：命中目标

此时你应该知道你命中了哪个 PID。

1. 返回 RNG Reporter。
2. 主界面输入你的种子，并确保是在 Generation 5 的 PIDRNG abuse 页面。
3. 点击「Search」，寻找你的 PID。

```
提示：你也可以导出结果为 .txt 文档，然后用 `Ctrl + F` 查找 PID。
```

4. 确定你从按下 `A` 后实际推进了多少帧，将这个数值从原本的目标帧数中减去，得到新的目标帧。
5. 尝试命中这个新帧数。如果没命中：
6. 请重复上述步骤。
7. 有可能需要多次尝试，但最终一定可以成功。
8. 推荐记录你尝试过的帧数以及你按下 `A` 的实际帧数。

![成功！](/images/Black-and-White/Roamer/Success.png)

## 恭喜！

使用大师球捕捉你的游走宝可梦，或使用锁定技能如黑色目光、踩影、影子标签等手段。然后在群里或朋友面前炫耀你的成果吧！
