```
与普通孵化乱数不同，此方法使用了 Masuda 法或闪耀护符，目标蛋只需乱数一次，但等待目标帧的时间可能更长。
```

## 所需工具

- [安装了 PokeReader 的 3DS](/install-pokereader)
- [3DSRNGTool](https://github.com/wwwwwwzx/3DSRNGTool/releases)

## 第一步：输入乱数信息

1. 选择游戏版本并输入 TSV（训练家异色值）。
2. 输入初始种子（Seed）。
3. 如果你拥有闪耀护符，请勾选「闪耀护符」。

```
如果你想生成一个不属于你的 ESV（蛋异色值）的蛋，请点击「TSV列表」来添加额外的 TSV，然后勾选「其他TSV异色」。

记得在右上角输入你自己的 TSV，这样才能正确判断哪些帧是闪。
```

## 第二步：输入父母信息

根据你所使用的父母宝可梦输入信息。如果父母是不同语言版本，请勾选「Masuda Method」。

```
判断 Masuda 法是否成立的是语言，而不是地区。即便两个宝可梦来自同一个国家，只要语言不同，就成立。
```

如果是百变怪和无性别宝可梦配对，系统会视百变怪为「母方」。

## 第三步：孵蛋seed、筛选条件与目标帧

1. 输入你当前游戏中的孵蛋seed。
2. 在筛选条件中填写你想要的蛋的属性。
3. 若只想查找异色，请勾选「仅异色帧」。
4. 在游戏中按下 `Start + Select` 暂停游戏。
5. 填写当前帧数，并扩大「帧数范围」以查找更多结果。
6. 若你是第一次进行操作，建议选择「接受」（接受第一个蛋）来确认帧数-1 的数据是否正确。如果选择「拒绝」，"考虑延迟" 计数器将重置为 0。
7. 点击「计算」开始搜索目标帧。如果无结果，请增大帧范围并重新计算。

## 第四步：乱数目标蛋的属性

1. 选中目标帧。
2. 若你需要从奇数帧跳到偶数帧，或反过来，只需按`Start`解除暂停后存一次档。

记住：

- 战斗外每步为 2 帧跳动。
- 游戏运行过程中帧状态要么全是偶数，要么全是奇数。
- 存档一次可切换帧奇偶性。
- 第一个拿到或拒绝的蛋「不是」你乱数的目标蛋。
- 蛋的属性在前一个蛋被处理（拿取/拒绝）时就已决定，之后不会改变。
- 换句话说：处理当前蛋，会决定下一个蛋。
- 3DSRNGTool 中显示的帧数-1 是当前蛋的属性。

## 第五步：乱数蛋的属性（不包含 PID / ESV）

1. 在游戏中与培育员对话，等待出现以下对话：
   - 接受蛋时（ORAS）："你会想要它的，对吧？"
   - 接受蛋时（XY）："你想要它，对吧？"
   - 拒绝蛋后（ORAS / XY）："那我就先保管好啦，谢谢你啦！"
2. 当你接近目标帧时，按下 `Start + Select` 暂停游戏，再使用 `Select` 每次推进一帧。
3. 当达到目标帧时，按住 `A` 同时解除暂停并领取/拒绝蛋。
4. 当前操作完成后，下一颗蛋的属性就会与帧数-1 匹配。

拿到下一颗蛋后，查看或孵化即可验证。恭喜你！

## 常见问题排查

- 确认 3DSRNGTool 右上角的 TSV 与主视图的 TSV 相同。
- 如果你领取的第一颗蛋的属性与帧数-1 不符，确认你是否使用的是 ORAS 正确的育婴员对话界面。
- 每次领取/拒绝蛋之后，孵蛋seed应会发生变化。如果没有，请确保你已使用最新版本的 PokeReader，且游戏为最新版。
