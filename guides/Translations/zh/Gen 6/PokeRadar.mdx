## 所需工具

- [PokeReader](/install-pokereader)
- 大量驱虫喷雾，用于避免普通野生宝可梦遭遇

## 步骤 1：前期准备

1. 确保你已安装并运行 PokeReader。
2. 打开 PokeReader，并按下 `X` 与 `Y` 将其锁定在屏幕上。
3. 使用 PokéRadar，并遭遇你想要进行异色乱数的宝可梦。
4. 击倒或捕捉该宝可梦，使连锁数变为 1，并锁定宝可梦种类。
   - 连锁数为 0 的草丛判定方式与连锁数为 1 及以上不同。你必须通过随机遭遇或使用野生乱数来获取目标种类。

```
注意：请避免使用轮滑、进入错误种类的草丛，或踩入空草丛，这些都会中断连锁。
```

## 步骤 2：输入数据

1. 打开背包以暂停 TinyMT 的推进。
2. 将以下从 PokeReader 中读取的数据输入到本页面底部的乱数工具中：
   - TinyMT Seed
   - 当前推进帧数
   - 最大推进帧数（例如 20000）
   - 队伍中的宝可梦数量
   - 当前连锁数
3. 点击 "Generate" 以显示可能的结果。

当你点击工具中的某一行时，下方会显示如下信息：

- **Sparkles**：异色宝可梦草丛
- **绿色对勾**：安全草丛，可继续连锁
- **红色叉号**：危险草丛，会中断连锁（错误种类或空草丛）

## 步骤 3：推进 TinyMT

1. 如果 PokéRadar 电量不足，请行走以进行充能。
2. 使用宝可拍拍乐来快速推进 TinyMT。
3. 当接近目标推进帧数时，打开背包以暂停推进。
4. 通过以下方式推进到目标帧数：
   - 开关经验分享，每次会按队伍中宝可梦数量推进 3 帧
   - 对宝可梦开始教学招式机并立刻退出，可推进 1 帧

## 步骤 4：寻找异色草丛

1. 达到目标推进帧数后，使用 PokéRadar。
2. 在不中断连锁的前提下，走入正确的草丛。
3. 捕捉你的异色宝可梦！

恭喜，你已经成功完成一次 PokéRadar 异色乱数！
