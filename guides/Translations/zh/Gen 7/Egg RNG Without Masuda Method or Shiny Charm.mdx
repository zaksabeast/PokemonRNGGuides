## 与使用不同语言法和/或闪耀呼符有何不同？

使用不同语言法和/或闪耀呼符时，蛋的异色值（ESV）是按固定模式生成的，唯一能控制帧数的方式是选择接受或拒绝蛋。但不使用这些时，ESV 是在你接受蛋的那一刻才生成的，因此任意帧都有可能生成任意 ESV。

```
说明：ESV决定蛋是否异色。当 ESV 与训练家的 TSV（训练家异色值）相同时，孵出的宝可梦就是异色。
```

## 所需工具

- 一台安装了 [PokeReader](/install-pokereader) 的 3DS
- [3DSRNGTool](https://github.com/wwwwwwzx/3DSRNGTool/releases)

---

## 第一步：设置 3DSRNGTool

### 在 3DSRNGTool 右上角：

1. 输入你所玩的游戏版本和 TSV。
2. 输入初始种子。
3. 取消勾选 “闪耀护符” 选项。使用此方法时不能持有闪耀呼符。

```
如果你要乱数目标不是自己的 ESV，请点击 "其他TSV异色" 并输入目标 TSV。但你自己的 TSV 仍需保留在右上角，否则会导致结果不正确。
```

### 父母信息部分：

根据你使用的亲代宝可梦填写：

- 确保未勾选 “父母语言不同”。此方法不能使用不同语言亲代。
- 检查父母语言是否相同，以避免后续问题。

```
说明：所谓“不同语言法”，指的是两只宝可梦语言不同，地区无关。
```

百变怪与无性别宝可梦配对时，百变怪视为雌性，否则为与另一只亲代相反性别的宝可梦。

关于岩狗狗（黄昏）孵蛋：

- 若特性为“我行我素”，设置特性为 1/2/H 都不会影响结果。
- 若不是我行我素，则可能是 [1]锐利目光，[2]干劲，[H]不屈之心。

### 当前状态部分：

1. 在“当前状态”中输入当前蛋的种子。
2. 暂时不要勾选 “主乱数蛋 (PID)”。
3. 在“个体筛选”中填写你希望孵出的宝可梦条件。
4. 即使想要异色，也不要勾选 “仅异色帧”。我们之后会另外进行 ESV 乱数。
5. 起始帧设为 “0”。
6. 点击 “计算”。

## 第二步：寻找目标帧

你可以选择任何列出的帧。较低的帧通常意味着更少的接收/拒绝操作。

1. 右键点击你想要的行，选择 “将该帧设为目标”。
2. 点击 “最短路径” 然后点 “计算”，会计算出最短的接受/拒绝路径。
3. 按照该路径顺序接收或拒绝蛋。顺序错误将导致种子错乱。不要提前接收目标蛋（即路径中最后一个蛋）。
4. 得到目标蛋种子后，继续下一步。

## 第三步：乱数目标蛋的 ESV

1. 接受蛋前站在培育员面前存档，以防失误。
2. 与培育员对话并开始接蛋流程。
3. 当出现“是否接收蛋”的选项时，使用 `Start + Select` 暂停游戏。
4. 通过以下步骤创建时间线以锁定目标 ESV：
   - 在 “当前状态” 中勾选 “主乱数蛋(PID)”。
   - 点击齿轮图标重置筛选条件。
   - 参考 [时间线创建指南](/retail-usum-timeline) 创建时间线。
   - 时间线用于识别由于 NPC 影响后仍可命中的帧。
5. 时间线创建后，搜索目标帧：
   - 若要给其他人乱数异色蛋，请点击 "TSV列表"，输入 TSV 并勾选 "其他TSV异色"。
   - 勾选 “仅异色帧”。
   - 点击 “计算” 查找符合条件的帧。
   - 选择任意一个蓝色高亮帧（帧数越低越省时间）。
6. 推进到对应帧数后，按 `A` 接受蛋。

恭喜！你现在应该成功拿到目标蛋，且已乱数出想要的 ESV。如果失败，只需读取接蛋前的存档重新尝试。
