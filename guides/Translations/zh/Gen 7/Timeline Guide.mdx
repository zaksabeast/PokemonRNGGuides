## 所需工具

- 一台安装了 [PokeReader](/install-pokereader) 的 3DS
- [3DSRNGTool](https://github.com/wwwwwwzx/3DSRNGTool/releases)

## 第一步：设置 3DSRNGTool

只有在当前区域存在一个或多个 NPC 时才需要创建时间线。如果是 0 个 NPC 的区域，则不需要时间线。

1. 输入你所玩的游戏版本和 TSV。
2. 输入初始种子，可在主乱数视图中找到 `Seed:` 一栏。
3. 如果持有闪耀呼符，勾选闪耀护符选项。

```
说明：在第七世代中，NPC 会推动帧数。通常每个 NPC 会使帧数前进 1，因此一个区域中如果有 4 个 NPC，帧数一般每次前进 5 帧。像洛托姆或主角眨眼等也会影响帧数。如果能正确创建时间线，就能精准预测帧数，避免因 NPC 干扰错过目标帧。
```

## 第二步：创建时间线

1. 在 3DSRNGTool 中勾选 “仅准确帧”。
2. 输入当前区域的 NPC 数量。请使用 PokeReader 确认该数值。
3. 游戏中推进至你要乱数的宝可梦的最终画面，在帧数范围中填写当前帧。
4. 点击 “计算”，然后推进至标记列中为 “-” 的任意一帧。这些为“准确帧”，可用来进行精准帧数判断。
5. 按 `Start` 解除暂停来推进帧数，接近目标帧后可使用 `Start + Select` 暂停，再用 `Select` 单步推进。

## 第三步：确认时间线与 NPC 数量

1. 达到任一准确帧后，在 3DSRNGTool 中输入当前帧，勾选 “创建时间线”，再点击 “计算”。
2. 使用 `Select` 键多次单步推进，并与 3DSRNGTool 中显示的后续帧进行比对。
   - 如果一致，说明 NPC 数正确。
   - 如果不一致，说明时间线错误，需重新创建。

时间线制作是否正确，通常在此阶段就能发现。请确认初始种子是否正确，并确保在制作时间线期间 PokeReader 中的 NPC 数量没有发生变化。

```
注意：制作时间线过程中请勿调整筛选器。此时仅应勾选 “仅准确帧”。若更改了其他筛选选项，请从头开始。
```

```
注意：可通过以当前帧为基准重新制作时间线，再搜索目标帧，以确认目标帧是否仍在时间线范围内。若目标帧不在结果中，说明时间线已偏移。
```
