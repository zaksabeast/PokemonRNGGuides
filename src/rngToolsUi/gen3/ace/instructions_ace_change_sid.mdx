
## Change SID with ACE
<br />
The following steps will change your SID to ensure your target Pokémon will be shiny.

1. Add to your party a glitch Pokémon capable of triggering ACE (ex: 0x611 Egg or Stable Species).
   - If you don't own such glitch Pokémon, check [Obtaining 0x0611 for use in Arbitrary Code Execution](https://e-sh4rk.github.io/ACE3/611.pdf).
1. Change the box names for the ones displayed below.
1. Save the game.
1. Trigger ACE.
   - If using 0x611 Egg, hatch the egg.
   - If using Stable Species, check the Pokémon summary.
1. After performing ACE, the Pokémon certificate should be displayed indicating that the trick worked. If it is displayed, move on to the next step.
1. If it is not displayed or if the game crashes, restart the game and make sure the box are named properly before trying again.
