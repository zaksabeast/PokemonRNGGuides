《太阳/月亮》闯入对战呼唤乱数教程

所需工具

  • 已安装PCalc的3DS (PCalc安装教程)
  • 3DSRNGTool

推荐阅读

  • 闯入对战呼唤概率表
  • 《太阳》遭遇表

注意：以下步骤基于《究极之日/究极之月》，但《太阳/月亮》的闯入对战 乱数操作完全相同。只需按本教程设置你的配置，无需担心细微差异。

闯入对战呼唤乱数机制解析

闯入对战呼唤乱数涉及两个关键过程：
  • G7 SFMT (32位)：控制闯入对战呼唤的乱数。决定遭遇槽、呼唤成功率及等级。
    • 通过此乱数可遭遇特定宝可梦（如1%出现率的暴飞龙）。
  • 主乱数：决定个体值（IVs）、性格、性别及PID（包含异色判定）。
闯入对战呼唤分为两类：
  1. 场上存在两只宝可梦，其中一只濒死。
  2. 最后一次呼唤失败，需使用道具触发新呼唤（稳定性更高，适合乱数操作）。
闯入对战呼唤是获取异色宝可梦的理想方式：当连锁达31次且拥有闪耀护符时，命中PID重抽区间（13-15帧）的概率大幅提升。连锁长度也影响隐藏特性的出现概率。可定位特定帧，但稳定性较低。
操作提示：开始乱数前请建立闯入对战连锁。选择低血量宝可梦呼唤以及使用胆怯球以最大化呼唤成功率。

步骤1：配置3DSRNGTool

  1. 输入游戏版本及TSV（在PCalc中按Start + ↑，TSV显示于游戏视图窗口）。
  2. 输入初始种子，位于游戏视图窗口的"Init Seed"处。
  3. 若拥有闪耀护符，勾选闪耀护符选项。

步骤2：遭遇闯入对战宝可梦并设置乱数工具

  1. 在地图上移动/甜甜蜜等方式或通过乱数遭遇正确的闯入对战呼唤宝可梦。
    • 第七世代野外乱数教程详见此处。
  2. 进入对战界面后，打开3DSRNGTool的"工具"菜单，选择"多功能乱数工具"。将乱数类型设为G7 SFMT (32bit)并切换至闯入对战标签页。
  3. 在PCalc中按B + ↑开启扩展游戏视图。按↓ + B重置闯入对战信息。
    • 若闯入对战初始种子显示全0，需在战斗第一个回合按↓ + B刷新。
  4. 将PCalc中的闯入对战初始种子填入3DSRNGTool的"Seed"字段。
  5. 查询呼唤概率并填入工具
    • （参考此表）。
  6. 完善闯入对战标签页的其他信息：
    • 设置当前呼唤者的HP值。
    • 若使用胆怯球则勾选对应选项。
    • 天气影响存在时勾选"天气"。
    • 若当前宝可梦持有威吓、紧张感或压迫感特性，勾选"威吓"。
    • 忽略"效果绝佳"选项（实际效果可能低于预期）。
注意：只要相同的宝可梦在呼唤，就请检查"相同呼叫者"选项。如果最初的宝可梦倒下了，请取消勾选该选项。
  1. 点击"计算"并分析结果：
    • 结果字段含义如下：
索引 | 推进值 | 闯入对战信息 | 随机数# 闯入对战信息：是否呼唤？是否呼唤成功？/ 首位特性 / 遭遇槽 / 等级 / 携带道具 / 满IV项 / 特性
  • 索引：闯入对战索引（从0开始计数）。
  • 推进值：本回合后推进的索引（呼唤成功时更高）。
  • 闯入对战信息：呼唤详情：
    • 第一项：是否呼唤成功？（O=是，X=否）
    • 第二项：是否出现新宝可梦？
    • 第三项：首位宝可梦特性是否生效？
    • 第四项：遭遇槽（决定天气、1%概率宝可梦等）。
    • 第五项：宝可梦等级。
    • 第六项：携带道具。
    • 第七项：满IV的项数。
    • 第八项：特性类型（普通/隐藏）。
  • 乱数值#：该帧对应的闯入对战当前种子。
注意：部分字段可能未启用，但含义应可直观理解。

步骤3：推进闯入对战索引

理想的操作时机是当连锁较长时（高V、目标宝可梦出现概率提升）：
  1. 设置闯入对战环境（樱子果、收获特性、胆怯球、同步特性等）。
  2. 通过PCalc扩展视图查看当前闯入对战索引。
  3. 每回合结束后更新3DSRNGTool信息：
    • 在辅助工具中更新：连锁长度、闯入对战索引、上次呼唤失败。
    • 在主窗口中更新：主乱数帧数、连锁长度。
  4. 输入数据后，预判下回合呼唤结果：
    • 若闯入对战信息前两项均为O，则呼唤成功。
    • 否则，根据"推进值"确定下一有效呼唤索引。
    • 若上回合呼唤失败，下次搜索前务必勾选"最后呼唤失败"。
    • 对成功呼唤的帧，可勾选"只显示成功"后点击"计算"筛选。
  5. 当命中呼唤成功索引（前两项均为O），用此索引校准主乱数延迟值。
    • 可通过使用胆怯球或击倒非呼唤者宝可梦推进闯入对战帧。
关键验证：比对PCalc中的闯入对战当前种子与3DSRNGTool中的"随机数#"是否一致。若不符，需同步调整多功能乱数工具的种子。闯入对战索引推进可能不稳定（尤其钓鱼闯入对战）。

步骤4：确定主乱数延迟值

配置3DSRNGTool主窗口

  1. 切换至"野生乱数"标签页，勾选"闯入对战"并选择正确地点。
  2. 根据游戏内时间选择"白天/夜晚"。
  3. 将辅助工具中的闯入对战初始种子填入"Seed"框。
  4. 将辅助工具的当前闯入对战索引填入"index"框。
  5. 若首位宝可梦有相关特性，正确设置"队首"选项。
  6. 按辅助工具显示输入连锁长度。
  7. 若存在天气影响，选择正确的"天气"选项。

命中目标宝可梦的主乱数帧

  1. 在3DSRNGTool主窗口，将当前主乱数帧设为起始帧。
  2. 通过PCalc查看当前主乱数帧：按Start + ↑打开游戏视图
  3. 再按Start + Select暂停游戏，查看"Frame:"后的数值。

选择命中任意主乱数帧以进行延迟校准

  1. 右键点击目标帧，选择"将该帧设为目标"。
  2. 根据场上宝可梦数量决定最终操作时机：
    • 两只在场：延迟较长，最终操作为选择目标时按A。
    • 仅一只在场：延迟较短，最终操作为对胆怯球按A选择"使用"。

确定实际命中帧

新宝可梦出现后，需通过其信息反推命中帧：
  1. 将"延迟"设为0，根据宝可梦信息搜索实际命中帧，基于目标帧的偏差/F值调整。
  2. 在"考虑延迟"框中输入按下A时的帧数。
  3. 若无结果，可能是闯入对战帧输入错误。请重新核对3DSRNGTool数据（尤其是闯入对战索引和随机数#）。
提示：如果您无法找到所获得宝可梦的个体值，请再次确认 SOS 框是否填写正确。若 SOS 框填写有误，则无法显示所获得宝可梦的正确数值。
提示：宝可梦信息对应帧 - 目标帧 = 延迟值
操作技巧：按Start + ← 开启野生宝可梦视图，Select + → 切换查看目标。

步骤5：重复校准以确定稳定延迟值

  1. 重复步骤3-4多次并记录结果。
  2. 当能稳定命中异色帧区间时，即可遭遇最终目标（建议在31连锁时追求高V和异色）。
  3. 选择最常见或中位延迟值：
    • 例如延迟值为56, 58, 62时，选择58更安全（易命中帧区间）。

步骤6：获取目标宝可梦

  1. 在"闯入对战2"标签页筛选目标：设置遭遇槽、同步成功及隐藏特性需求。
  2. 返回"闯入对战"标签页完善信息，筛选成功呼唤索引：
    • 勾选只显示成功确保仅显示呼唤成功索引。
  3. 通过筛选条件遭遇目标索引：
    • 需确保闯入对战信息前两项为OO。
进阶技巧：尝试遭遇1%稀有宝可梦时，接近目标索引时可利用威吓特性提高成功率。
  1. 命中目标闯入对战索引后，再命中主乱数目标帧：
    • 确保3DSRNGTool主窗口信息正确。
    • 务必根据闯入对战类型设置正确延迟值！
  2. 设置宝可梦属性筛选条件：
    • 注意个体值限制（参考辅助工具第七项，如VVXVVX表示特定项满IV）。
  1. 将游戏操作推进至最终指令前。
  2. 接近目标帧时暂停游戏，推进至帧区间中部后执行最终操作。
 恭喜！现在你应已获得目标宝可梦。

未获得目标时的排查步骤

  1. 检查两工具的所有输入项（初始种子、闯入对战索引等）。
  2. 重新按教程逐步操作。
  3. 不同闯入对战类型可能有差异延迟值，选择最常见或中位值：
    • 例如延迟56, 58, 62时选择58更稳妥（易覆盖帧区间）。
